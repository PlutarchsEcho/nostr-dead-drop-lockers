## Grant Application Tips

**What Funders Want to See:**
- ✅ Open source (GPL/MIT)
- ✅ Working prototype
- ✅ User adoption (even small)
- ✅ Clear roadmap
- ✅ Team credentials
- ✅ Privacy/censorship resistance angle
- ✅ Global impact potential

**Our Advantages:**
- Open hardware standard
- Bitcoin + Lightning native
- Nostr integrated
- Privacy-preserving
- Can be deployed anywhere
- No KYC required

**Application Materials to Prepare:**
- [ ] One-page project summary
- [ ] Demo video/video walkthrough
- [ ] GitHub repo with active commits
- [ ] User testimonials (even from alpha testers)
- [ ] Budget breakdown
- [ ] 12-month roadmap
- [ ] Team bios
